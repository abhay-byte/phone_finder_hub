# Ultra-Extensive Phone Scoring System (UEPS-45)

This is an advanced, enthusiast-level scoring system designed for the 2026 flagship landscape. It evaluates **40 distinct criteria** across 7 categories.

**Total Possible Score: 245 Points**

## 1. Categories & Criteria

### A. Build & Durability (30 pts)
1.  **Frame Material**: Aluminum (+3), Titanium/Stainless (+5), Plastic (+0).
2.  **Back Material**: Glass/Ceramic (+5), High-grade Fiber (+3), Plastic (+1).
3.  **Front Glass**: Gorilla Glass Victus 2 or better (+5).
4.  **IP Rating**: IP68 (+5), IP69/K (High-pressure/temp) (+10).
5.  **Bezel Size**: Under 1.5mm (+5).

### B. Display Tech (30 pts)
6.  **Panel Type**: LTPO AMOLED/OLED (+5), Standard AMOLED (+3), LCD (+1).
7.  **Refresh Rate**: 120Hz (+3), 144Hz-165Hz (+5).
8.  **Peak Brightness**: >2000 nits (+5), >4000 nits peak (+10).
9.  **Resolution**: 1.5K/2K (+5).
10. **Eye Care**: High-frequency PWM Dimming (>2160Hz) (+5).
11. **Color Depth**: 10-bit or 12-bit support (+5).
12. **Screen-to-Body Ratio**: >90% (+5).

### C. Processing & Memory (30 pts)
13. **Processor Tier**: Snapdragon 8 Elite / Dimensity 9400 (+10).
14. **Cooling System**: Large Vapor Chamber (+5), Active Inbuilt Fan (+10).
15. **RAM Tech**: LPDDR5X (+3), LPDDR5X Ultra/LPDDR6 (+5).
16. **Storage Tech**: UFS 4.0/4.1 (+5).
17. **SD Card Slot**: Available (+5).
18. **RAM Options**: Availability of 16GB/24GB variants (+5).

### D. Power & Charging (30 pts)
19. **Battery Capacity**: >5000mAh (+5), >7000mAh (+10).
20. **Wired Speed**: >100W (+5).
21. **Wireless Charging**: Available (+3), >50W Wireless (+5).
22. **Reverse Wireless**: Available (+5).
23. **Bypass Charging**: (Powering chip directly for gaming) (+5).
24. **Reverse Wired Charging**: (Charging other phones via cable) (+5).

### E. Camera Mastery (30 pts)
25. **Main Sensor Size**: 1-inch type (+5).
26. **Zoom Hardware**: Periscope Telephoto (+5).
27. **Video Max**: 8K support (+2), 4K/120fps (+5).
28. **Front AF**: Autofocus on selfie camera (+3).
29. **Additional Sensors**: Color Spectrum / Flicker sensor (+5).
30. **OIS**: Optical Image Stabilization on at least 2 lenses (+5).

### F. Connectivity & Ports (25 pts)
31. **USB Speed**: USB 3.2 Gen 1/2 (+5).
32. **Video Out**: DisplayPort Alt Mode support (+5).
33. **IR Blaster**: Included (+5).
34. **NFC**: Included (+2).
35. **Bluetooth/Wi-Fi**: BT 6.0 / Wi-Fi 7 (+3).
36. **Satellite Link**: Emergency SMS via Satellite (+5).

### G. Audio & Extras (15 pts)
37. **Headphone Jack**: 3.5mm Port (+5).
38. **FM Radio**: Included (+5).
39. **Fingerprint**: Optical (+3), Ultrasonic (+5).
40. **Haptics**: X-axis Linear Motor / High-grade vibration (+5).

### H. Developer Freedom & Emulation (55 pts)

**Bootloader Unlockable:**
*   Official unlock supported (+10)
*   No official unlock (+0)

**OS Openness:**
*   Near-AOSP / Minimal restrictions / Easy root (+10)
*   Moderately restricted (+5)
*   Heavily restricted OEM skin (+0)

**Turnip / Mesa Custom Driver Support (Adreno only):**
*   Full latest Mesa Turnip support (actively maintained & stable) (+20)
*   Stable but slightly outdated builds (+15)
*   Partial / unofficial ports (+8)
*   Not supported (+0)

**GPU Emulation Compatibility Tier (20 pts total):**
*   **Adreno GPUs:**
    *   Adreno 8xx Elite-class (full modern driver compatibility) (+20)
    *   Adreno 7xx strong support (+16)
    *   Adreno 6xx moderate support (+10)
*   **Immortalis GPUs:**
    *   Immortalis G925 / G720 high-tier support (+14)
*   **Mali GPUs:**
    *   Mali Valhall (G710 / G715) (+10)
    *   Older Mali (G78 and below) (+6)

**Custom ROM Community Support:**
*   Major active ROM ecosystem (LineageOS, Pixel Experience, etc.) (+10)
*   Limited unofficial builds (+5)
*   None (+0)
